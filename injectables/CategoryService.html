getCategories
getCategories(limit: number, hierarchyIdList?: Array)

Returns all the categories available to the user. Will translate the categories based on the user locale.

Example:

import { CategoryService, Category } from '@apttus/ecommerce';
import { Observable } from 'rxjs/Observable';

export class MyComponent implements OnInit{
    categoryList: Array<Category>;
    categoryList$: Observable<Array<Category>>;

    constructor(private categoryService: CategoryService){}

    ngOnInit(){
        this.categoryList$ = this.categoryService.getCategories();
        // or
        this.categoryService.getCategories().subscribe(c => this.categoryList = c);
    }
}
Parameters :
Name Type Optional Default value Description
limit number No 1000

the maximum number of categories to return. Note: more than 1000 may cause issues.
hierarchyIdList Array<string> Yes
Returns : Observable<Array<Category>>

an array of categories in an observable. Use getCategoryTree() if you need a hierarchy structure.
getCategoryBranch
getCategoryBranch(categoryId: string)

getCategoryBranch takes a single category id and returns an array containing the category and its immediate parents up to the root. The first element in the array is the root and the last is the category specified in the argument. Designed for breadcrumbs.

Example:

        // or

    }
}
Parameters :

getCategoryBranchChildren
getCategoryBranchChildren(categoryIdList: Array)

getCategoryBranchChildren takes a single category id and returns an array containing the category and all of its children down to the leaves. Note: does not maitain relationships with ancestor / child categories using _children property. This simply is a list of all categories under a given category.

getCategoryBranchForProduct
getCategoryBranchForProduct(product: Product)

Similar to getCategoryBranch, but takes a product for an argument and returns the array of categories leading to that product.

Example:

import { CategoryService, Category, Product, ProductService } from '@apttus/ecommerce';
import { Observable } from 'rxjs/Observable';

export class MyComponent implements OnInit{
    categoryList: Array<Category>;
    categoryList$: Observable<Array<Category>>;

    constructor(private categoryService: CategoryService, private productService: productService){}

    ngOnInit(){
        this.categoryList$ = this.productService.getProductsByCode(['xxx'])
                                .map(p => p[0])
                                .flatMap(product => this.categoryService.getCategoryBranchForProduct(product));
        // or
        this.productService.getProductsByCode(['xxx'])
                                .map(p => p[0])
                                .flatMap(product => this.categoryService.getCategoryBranchForProduct(product))
                                .subscribe(categoryList => this.categoryList = categoryList);
    }
}
Parameters :
Name Type Optional Description
product Product No

an instance of the product to get the category branch for.
Returns : Observable<Array<Category>>

a single dimensional array containing the product's category and its immediate parents up to the root in an Observable

getCategoryTree
getCategoryTree(categoryNames?: Array, hierarchyIdList?: Array)

Returns the categories as a tree structure. Will return all categories or those specified in the categoryNames argument. Adds an additional attribute to all categories returned called '_.children'. This attribute contains an array of child categories.
