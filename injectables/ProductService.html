Public getAggregatesForCategory
getAggregatesForCategory(categoryId: string, categoryFilter: Array, priceTier: PriceTier, adtlConditions: Array)

Method is similar to getProductsByCategory, however it returns the aggregate data (i.e. total records) instead of the sobject results.

Example:

import { ProductService, Product } from '@apttus/ecommerce';
import { Observable } from 'rxjs/Observable';

export class MyComponent implements OnInit{
    aggregateData: any;
    aggregateData$: Observable<any>;

    constructor(private productService: ProductService){}

    getAggregateDataForCategory(categoryId: string){
        this.productService.getAggregatesForCategory(categoryId).subscribe(p => this.aggregateData = p);
        // or
        this.aggregateData$ = this.productService.getAggregatesForCategory(categoryId);
    }
}
Parameters :
Name Type Optional Description
categoryId string No

the id of the category that you want to pull the products for.
categoryFilter Array<Category> No

a list of specific subcategories that can be used to further filter the list
priceTier PriceTier No

a price tier object representing the price range to filter by
adtlConditions Array<ACondition> No

an array of ACondition records that will be used in the where clause (i.e. [new ACondition(Product, 'ProductFamily', 'Equals', 'Robots')])
Returns : Observable<any>

Public getProductsByCategory
getProductsByCategory(categoryId: string, limit: number, pageNumber: number, orderBy: string, orderDirection: "ASC" | "DESC", categoryFilter: Array, priceTier: PriceTier, adtlConditions: Array)

A method to retrieve a list of products for a specific category. Provides a number of optional filters.

Example:

import { ProductService, Product } from '@apttus/ecommerce';
import { Observable } from 'rxjs/Observable';

export class MyComponent implements OnInit{
    productList: Array<Product>;
    productList$: Observable<Array<Product>>;

    constructor(private productService: ProductService){}

    getProductsByCategory(categoryId: string){
        this.productService.getProductsByCategory(categoryId, 10, 0, 'Name', 'ASC', null, null, [`ProductFamily = 'Sample'`, 'Active = TRUE']).subscribe(p => this.productList = p);
        // or
        this.productList$ = this.productService.getProductsByCategory(categoryId);
    }
}

Public globalFilter
globalFilter()

The global filter is for retrieving products from this service. The default filter is set to only get products that are active and have an active price list item associated with the current user's account.

Returns : Observable<Array<AFilter>>
