Public getSearchResults
getSearchResults(query: string, limit: number, pageNumber: number, orderBy: string, orderDirection: "ASC" | "DESC", categoryFilter: Array, priceTier: PriceTier, adtlConditions: Array)

The primary method that takes a search query string and returns a list of products. Uses standard Salesforce SOSL to retrieve results.

Example:

import { SearchService, SearchResults } from '@apttus/ecommerce';
import { Observable } from 'rxjs/Observable';

export class MyComponent implements OnInit{
    searchResults: SearchResults;

    constructor(private searchService: SearchService){}

    getSearchResults(query: string){
        this.searchService.getSearchResults(query).subscribe(
            results => this.searchResults = results,
            err => {...}
        );
    }
}
Parameters :

the query string to find products by
limit number No 10

the limit of the number of products to return (defaults to 10)
pageNumber number No 0

the offset of the products to return (i.e. for results 10-20, offset will be 10)

the product field with which to order the results by

the direct to order the results (ASC or DESC)

an array of category instances to further filter the results by
